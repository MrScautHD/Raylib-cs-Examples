## Getting Started

### Building

1. `git clone https://github.com/ChrisDill/Raylib-cs-Examples.git`

2. `cd Raylib-cs-Examples`

3. `dotnet build`

### Running

1. `dotnet run -p Examples/Examples.csproj`
